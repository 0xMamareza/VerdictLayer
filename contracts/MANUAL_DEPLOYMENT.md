# Manual GenLayer Studio Deployment

1. Open https://studio.genlayer.com/contracts.
2. Create or open a contract.
3. Paste `contracts/verdict_layer.py`.
4. Deploy with no constructor parameters.
5. Call `get_contract_version`.
6. Call `submit_claim_verdict` with test values.
7. Call `get_latest_claim_verdict`.
8. Screenshot the contract address, successful deploy, and method results.
