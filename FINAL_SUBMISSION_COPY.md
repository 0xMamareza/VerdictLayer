# Consenza Final Submission Copy

## Product Name

Consenza

## One-Line Description

Onchain decisions for claims, tasks, and disputes.

## Short Description

Consenza supports Claim decisions, Task reviews, and Dispute resolutions through wallet-signed GenLayer workflows, then returns typed read-after-write results to the interface.

## What It Does

- Claim Decisions evaluates Web3 claims against submitted source URLs.
- Task Reviews examines completed builder or bounty work against requirements and proof.
- Dispute Resolutions addresses evidence-based disagreements using both sides' claims and a decision rule.
- Each production workflow requires explicit wallet approval, writes to the deployed GenLayer contract, and reads the updated result back into the interface.
